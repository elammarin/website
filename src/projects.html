Web Service
API REST FlopBox

Flopbox est une API REST qui permet de manager différents serveurs FTP ainsi que leurs fichiers. Ajout/suppresion de serveurs à manager, modification/céation/suppression des dossiers et/ou des fichiers du serveurs renseigné qui a été ajouté au préalable. Nécessaire au projet agent FlopBox.

Web Service
Agent FlopBox

Un client permettant de se connecter à differents serveurs FTP en utilisant l'API REST Flopbox. Permet d'ajouter/supprimer des serveurs FTP, lister les serveurs ajoutés, permet de gérer l'arborescence de fichiers des ces serveurs à travers plusieurs commandes à taper dans un shell.
